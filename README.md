🌍 Terraform Infrastructure Project
📌 Overview

This is uses of Terraform to provision and manage cloud infrastructure in a scalable and automated way.

It follows Infrastructure as Code (IaC) principles to ensure:

Reproducibility
Version control
Automation
Consistency across environments
🏗️ Architecture

This project provisions the following resources:

Resource Group
Virtual Network (VNet)
Subnets
Network Security Group (NSG)
Storage Account
Virtual Machine (Optional)
⚙️ Technologies Used
Terraform
Cloud Provider: Azure
Git & GitHub
CLI Tools
